Custom CSS theme builder

Click on the color wheel to choose a primary (1) and accent (2) color to preview the theme below.

When you’ve selected a color combination you like, either reference our hosted CSS or download the CSS by clicking the button in the middle of the color wheel.

You will need to include MDL’s JavaScript alongside your customised CSS to get the full experience. This is included in MDL's default Download from the Getting Started guide.

Reference the CSS files (MDL and Wijmo Material) from CDN:

<link rel="stylesheet" href="{{materialUrl}}" />
<link rel="stylesheet" href="{{wijmoUrl}}" />
