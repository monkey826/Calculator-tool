Chart Animation

This page shows how to play animation on FlexChart and FlexPie.

FlexChart

This view shows ChartAnimation for FlexChart.

You can use different animation modes for FlexChart by setting the animationMode property of the ChartAnimation.

The ChartAnimation class has a duration property that allows you to set the length of animation in milliseconds.

In this sample, maximum number of series is set to 5.

FlexPie

This view shows ChartAnimation for FlexPie.

You can use different animation modes for FlexPie by setting the animationMode property of ChartAnimation.

The ChartAnimation class has a duration property that allows you to set the length of animation in milliseconds.
