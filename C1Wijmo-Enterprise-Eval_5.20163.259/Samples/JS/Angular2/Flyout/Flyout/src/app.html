FlexGrid column header Flyouts

This sample defines a FlexGridFlyout class that monitors the mouse and displays a flyout element when the user hovers over a column header.

The flyout menu is just an ordinary div element. The sample populates it with some buttons and listens to click events to execute commands that apply to the column that owns the flyout currently visible.

Move the mouse over a grid column to see the flyout menu. Click any of the menu options to execute the command.
