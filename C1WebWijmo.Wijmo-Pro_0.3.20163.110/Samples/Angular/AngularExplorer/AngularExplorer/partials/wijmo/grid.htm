Wijmo Grid

The Wijmo grid widget allows users to interactively select, edit, sort, scroll through, filter, and group data. The grid is highly customizable and can be used as a building block for many types of specialized directives.

The grid below is bound to the list of airports used throughout this sample. The grid shows a list of airports grouped by state. The group headers show the number of airports in each state and the total air traffic in 2011.
