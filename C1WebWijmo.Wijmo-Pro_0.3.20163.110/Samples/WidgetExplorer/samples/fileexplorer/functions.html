The Wijmo FileExplorer widget provides many powerful functionalities such as creating new folder, delete, rename or copy a folder, you can peform these actions through the tool bar or context menu, you can also do this by keyboard and shortcut keys.

This demo shows you can enable/disable the Wijmo FileExplorer widget to do copy, create new items, or open a file by double click, etc.
